A refund request can be made before the official batch start date of the course.

2. How to Request a Refund

To request a refund, please email us at: selfcraftskills@gmail.com

Include the following details:

  • Full Name

  • Course Name
  • Razorpay Payment ID

3. Refund Processing

Approved refunds will be processed to the original payment method used at checkout via Razorpay.

Refunds are typically completed within 3-5 business days, depending on bank processing times.

4. Important Notes
